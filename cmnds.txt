git init => for creating the git of the current folder
git status => status of every file added or not
git add . => add every files
git add {filename} => add specific file name
git commit - m "msg" => commit with message
git log => shows every commit with head
git checkout {id} => all codes changed to that commit and head is changed
git switch - => undo checkout
git reser --hard {id} => all reset to this commit else deleted
git checkout -- . => reset to last saved commit
git checkout -b {branchname} => creates new branch
git checkout {branchname} => go to that branch
git branch => shows all branch
